# Structure Command

Organize content for presentations, documentation, or communication.

## Workflow

1. **Determine type**:
   - `pres` / `presentation` - Slide deck
   - `doc` - Technical documentation
   - `outline` - General outline

2. **Understand context**:
   - Topic
   - Audience
   - Goal/purpose
   - Constraints (time, length)

3. **Create structure** using structuring skill frameworks

4. **Present and iterate**

## Examples

```bash
/structure pres RAG Architecture Overview
/structure doc Project Kickoff Notes
/structure outline Q4 Planning
```

## Output

Numbered/bulleted structure ready to expand.

Ask: "Passt diese Struktur? Soll ich einen Bereich ausarbeiten?"
