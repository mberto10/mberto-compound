# KW Comment Template

Use this format for weekly status updates on YouTrack epics:

```markdown
**KW[XX] Update**

**Erledigt:**
- [Task 1 that was completed]
- [Task 2 that was completed]

**In Arbeit:**
- [Task currently being worked on]
- [Another task in progress]

**Blocker:**
- [Any blocking issues - omit section if none]

**Nächste Schritte:**
- [What's planned for next week]
- [Key milestones ahead]
```

## Example

```markdown
**KW51 Update**

**Erledigt:**
- RAG Pipeline v2 deployed to production
- Performance tests completed (95th percentile < 2s)
- Documentation updated in Confluence

**In Arbeit:**
- Fine-tuning embedding model for German content
- A/B test setup for new ranking algorithm

**Nächste Schritte:**
- Launch A/B test (KW52)
- Stakeholder demo scheduled for 20.12.
```

## Tips

- Keep bullet points concise (1 line each)
- Use German for consistency with team
- Link to relevant tickets/PRs where helpful
- Include metrics when available
