# Component Patterns

This document defines the standard UI components for FAZ prototypes and applications. 

*Note: All components must strictly adhere to the color and typography rules defined in [corporate-design.md](./corporate-design.md).*

## Buttons & Action Elements

### Article Tool Bar (Anhoren, Merken, Teilen, Drucken)
Utility actions associated with an article.
- **Icon + Text**: Links aligned horizontally.
- **Typography**: Suisse Regular/Medium, Set in White (on dark mode) or Black/Gray (on light).
- **Icons**: Clean, outlined, vector-style icons used alongside text.

### Badge / Kicker (Rubrik)
Used to denote the category or context of an article (e.g., "LIVEBLOG IRANKRIEG", "PROZESS IN OSLO").
- **Typography**: Suisse, Versalien (All-Caps), small size.
- **Layout**: Often accompanied by a colored dot (e.g., red for Liveblog) or an icon (e.g., F+ for premium content).
- **Color**: Gray or specific accent color.

## Cards & Teasers (Article Modules)

### Hero Teaser (Aufmacher)
The main article spotlight on a homepage or section.
- **Image**: Large, full-width or prominent placement.
- **Kicker**: Rubrik / Context (e.g., "BALANCEAKT IM OVAL OFFICE").
- **Headline**: Sang Bleu Kingdom, very large size, high contrast (White on dark background).
- **Teaser Text (Lead)**: Suisse or Sang Bleu, medium size, provides context.
- **Meta-Info**: Author names and comment counts placed at the bottom in a smaller size (Suisse).
- **Bookmark Icon**: A save/bookmark icon is typically placed in the top right corner relative to the text block.

### Standard Teaser (List elements)
Used for standard news roll elements below the hero.
- **Kicker**: F+ badge or red dot followed by Rubrik.
- **Headline**: Sang Bleu Kingdom, large size.
- **Teaser Text**: Suisse, descriptive summary.
- **Meta-Info**: Author and Location.

## Navigation & Header

### Main Header (Global)
- **Background**: Dark (Tiefblau or Dark Gray for dark mode settings).
- **Elements**: 
  - *Left*: Hamburger Menu (`☰`), Search Icon (`🔍`).
  - *Center*: FAZ Logo (klassischer Fraktur-Schriftzug).
  - *Right*: "Abo" Button (accented background, often Gold/Yellow for CTA) and User Profile Icon.
- **Sub-Navigation**: Horizontal list of sections (Politik, Wirtschaft, Finanzen, etc.) set in Suisse, White/Light text on dark background.
