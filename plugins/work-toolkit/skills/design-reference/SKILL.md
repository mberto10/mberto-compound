# FAZ Design Reference

This skill provides a structured reference for FAZ (Frankfurter Allgemeine Zeitung) corporate design, specifically tailoring patterns and guidelines for building prototypes, wireframes, and final web applications.

## How to use this skill

When working on any FAZ-related design, UI implementation, or wireframing task, consult the references in this directory:

- **[Corporate Design](./references/corporate-design.md)**: Brand colors, typography rules, spacing, and logo usage.
- **[Component Patterns](./references/component-patterns.md)**: Standardized UI components like buttons, forms, cards, and navigation.
- **[Layout Patterns](./references/layout-patterns.md)**: Grid systems, page structures, and responsive design rules.

## Design Philosophy

- **Authoritative & Clear**: The interface should reflect the high journalistic standards of FAZ.
- **Focus on Content**: Typography and spacing should prioritize reading experience above all else.
- **Restrained Accents**: Use the Störerfarbe (Gold) minimally to guide user attention to critical actions.

*Additional examples (wireframes, annotated screenshots) can be found in the `examples/` directory.*
