# Example Command

This is an example slash command that demonstrates the command structure.

## Purpose
This command shows you how to create custom slash commands for Claude Code.

## Instructions
When this command is invoked, greet the user and explain what commands can do:

- Commands are markdown files in the `commands/` directory
- They provide custom instructions and prompts for Claude
- Users invoke them with `/command-name`
- Commands can include context, examples, and specific behaviors

Provide a friendly greeting and offer to help with their task.
