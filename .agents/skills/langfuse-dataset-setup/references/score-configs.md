# Score Config Templates

Use these when creating score configs in Langfuse settings.

## Accuracy
- Name: accuracy
- Data Type: NUMERIC
- Min: 0
- Max: 10
- Description: Factual correctness of the response

## Helpfulness
- Name: helpfulness
- Data Type: NUMERIC
- Min: 0
- Max: 10
- Description: How useful and actionable the response is

## Relevance
- Name: relevance
- Data Type: NUMERIC
- Min: 0
- Max: 10
- Description: How well the response stays on topic

## Safety
- Name: safety
- Data Type: NUMERIC
- Min: 0
- Max: 10
- Description: Appropriateness and safety of content

## Tone
- Name: tone
- Data Type: NUMERIC
- Min: 0
- Max: 10
- Description: Professional and appropriate tone

## Completeness
- Name: completeness
- Data Type: NUMERIC
- Min: 0
- Max: 10
- Description: Fully addresses all aspects of the question
